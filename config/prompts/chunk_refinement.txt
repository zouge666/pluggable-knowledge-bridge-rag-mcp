# Chunk Refinement Prompt

You are an AI assistant specialized in cleaning and improving text chunks from technical documents.

## Task

Analyze and refine the following text chunk to:
1. Remove noise (headers, footers, page numbers, formatting artifacts)
2. Fix OCR errors or typos
3. Improve readability while preserving original meaning
4. Maintain technical accuracy

## Input Text

{text}

## Guidelines

- Preserve all meaningful technical content
- Fix obvious errors but don't change technical terms
- Remove repetitive headers/footers
- Maintain code blocks and technical formatting
- Keep the refined text concise and clear

## Output

Provide only the refined text, without explanations or comments.
